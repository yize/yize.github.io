ubuntu12.04phpstorm 5.0.4增加快捷启动

Mar 6th, 2013

在ubuntu上怎么创建phpstorm的快捷方式？(phpstorm.desktop) 其实很简单

phpstorm菜单 —> Tools —> Create Desktop Entry …

按下super(win)键，搜索下phpstrom试试，也可以直接把phpstorm固定在任务栏快速启动。
