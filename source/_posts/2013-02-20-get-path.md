linux下获取当前文件路径：
```sh
current_dir=$(cd "$(dirname "${0}")"; pwd)
echo current_dir
```
linux下获取当前执行路径：
```sh
if [ $(echo $0 | grep '^/') ]; then
    path=$(dirname $0)
else
    path=$(pwd)/$(dirname $0)
fi

echo $path
```
